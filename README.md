# Hellbound Hot Sauce

Premium artisan hot sauces and BBQ rubs handcrafted with the world's hottest peppers.

## About

This is the official website for Hellbound Hot Sauce, showcasing our premium line of artisan hot sauces and BBQ rubs. The site is designed for optimal SEO, local search visibility, and e-commerce integration with Shopify.

Developed by [Level 16 Media](https://level16media.com)

## Getting Started

### Prerequisites

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Development

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd hellbound-sauces

# Install dependencies
npm i

# Start the development server
npm run dev
```

### Build for Production

```sh
# Create optimized production build
npm run build

# Preview production build
npm run preview
```
